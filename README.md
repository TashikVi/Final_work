## Установка
* Установите [Node.js](https://nodejs.org/en/download/)
`brew install node`
* Установите [gulp] (https://gulpjs.com/)
* Установите зависимости проекта `gulp install`

## Для разработки:
gulp 

## Сборка:
gulp build
